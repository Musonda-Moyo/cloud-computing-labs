# Lab 03 – Basic Networking

## Objective
Understand basic cloud networking concepts including IP addressing, subnets, and security rules.

## Tools Used
- Cloud platform (AWS / Azure / GCP – practice environment)
- Linux
- Git & GitHub

## Steps
1. Created a virtual network (VPC / VNet)
2. Configured a subnet
3. Assigned private IP addresses
4. Configured basic firewall or security group rules
5. Verified connectivity using ping or SSH

## What I Learned
- Basic cloud networking concepts
- How subnets and IP addressing work
- How security rules control traffic
