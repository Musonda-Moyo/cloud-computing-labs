# Lab 01 – Virtual Machine Basics

## Objective
Learn how to create, configure, and manage a virtual machine in a cloud environment.

## Tools Used
- Cloud platform (AWS / Azure / GCP – practice environment)
- Linux
- Git & GitHub

## Steps
1. Created a virtual machine instance
2. Selected a Linux OS image
3. Configured CPU, memory, and storage
4. Connected using SSH
5. Verified system status with basic commands

## What I Learned
- How virtual machines work
- Basic Linux commands
- VM lifecycle (create, start, stop, delete)
